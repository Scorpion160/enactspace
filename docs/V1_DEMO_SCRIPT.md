# EnactSpace V1 - Script demonstration

Date: 2026-07-03

## 1. Introduction

Presenter EnactSpace comme le space numerique interne d'Enactus ESP: un outil unique pour organiser l'equipe, fluidifier la communication, suivre les projets et garder la memoire du club.

Message cle:

```text
EnactSpace permet a chaque Enacteur, Enacchef, Alumni ou candidat d'avoir une experience adaptee a son role.
```

## 2. Dashboard

Se connecter avec un compte Admin ou Team Leader.

Montrer:

1. Vue globale.
2. Cartes de synthese.
3. Acces rapide aux modules.
4. Difference attendue avec une vue membre simple.

## 3. Chat

Montrer:

1. Liste des conversations.
2. Conversation privee.
3. Groupe pole/projet.
4. Participants.
5. Messages, medias et etat de lecture si disponible.
6. Experience proche WhatsApp.

## 4. Posts

1. Fil d'actualite.
2. Creation post.
3. Commentaires.
4. Reactions.
5. Posts officiels et epingles.

## 5. Membres, poles et projets

2. Profil membre.

4. Liste poles.
5. Detail pole.
6. Liste projets.
7. Detail projet, equipe, budget et avancement.

## 13. Scenario multi-role

Enchainer rapidement:

1. Admin: vue globale.
2. SG: membres et presences.
3. Financier: finance.
4. Chef de pole/projet: perimetre limite.
5. Membre simple: espace personnel.
6. Alumni: acces limite.
7. Candidat: suivi candidature uniquement.

## 14. Conclusion

Insister sur:

1. Experience mobile-first.
2. Interface responsive.
3. Permissions par role.
4. Donnees centralisees.
5. Evolution prevue: QR code, NFC, Mobile Money, FCM, production.
